# Tao Science

Bilingual exploration of Taoist philosophy and modern science across five dimensions.

[![Live Demo](https://img.shields.io/badge/demo-tao--science.vercel.app-jade)](https://tao-science.vercel.app)

## Overview

A bilingual (English / Chinese) single-page site weaving Taoist cosmology together with contemporary scientific and philosophical frameworks across five dimensions:

1. **Human (人)** -- New Confucianism and the cultivation of self
2. **Earth (地)** -- Market Economy and material organization
3. **Heaven (天)** -- Dataism and information as cosmic principle
4. **Tao (道)** -- Evolutionism and the unfolding of complexity
5. **Nature (自然)** -- String Theory and the fabric of reality

Features an animated Taiji (yin-yang) symbol, sticky dimension navigation, and a grand comparison table synthesizing all five perspectives.

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript, HTML5 Canvas
- **Aesthetic:** Zen-void minimalism with jade accent color
- **Typography:** Cormorant Garamond (English) + Noto Serif SC (Chinese)
- **Navigation:** Sticky dimension sidebar for quick section jumps
- **Deployment:** Vercel
